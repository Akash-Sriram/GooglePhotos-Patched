# Google Photos Patched Releases

Automated releases for Google Photos patched with custom Morphe patches from [Akash-Sriram/De-Vanced](https://github.com/Akash-Sriram/De-Vanced).

## Features & Downloads
- Automatically checks APKMirror for new Google Photos (`nodpi`, `arm64-v8a`) releases every 6 hours.
- Compiles the latest patches from [De-Vanced](https://github.com/Akash-Sriram/De-Vanced).
- Patches the APK using Morphe CLI, aligns, and signs it.
- Downloads are available under the [Releases](https://github.com/Akash-Sriram/GooglePhotos-Patched/releases) section.
